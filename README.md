#centos_init_lnmp

> centos系统下的lnmp环境一键安装脚本。
> 支持在线下载安装和本地软件包安装。

> 离线的软件包在这个项目里面有（速度比官网要快些。。。）：[https://git.coding.net/xuanskyer/lnmp-packages.git](https://git.coding.net/xuanskyer/lnmp-packages.git)
> 下载后放到当前项目的packages目录下，修改对应的软件包名
